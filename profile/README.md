# StellarView

Open-source tools for exploring the [Stellar](https://stellar.org) network — ledgers, transactions, accounts, assets, and Soroban smart contracts, across Public, Testnet, and Futurenet.

🔭 **Explorer:** [stellarview.acachete.xyz](https://stellarview.acachete.xyz/) &nbsp;·&nbsp; 📚 **Docs:** [docs.stellarview.acachete.xyz](http://docs.stellarview.acachete.xyz/)

## Products

| Product | What it is | Links |
|---|---|---|
| [**StellarView Explorer**](https://github.com/StellarViewOrg/stellarview-explorer) | Web block explorer for Stellar and Soroban. Fast, real-time, multi-network. | [Live](https://stellarview.acachete.xyz/) |
| [**StellarView Indexer**](https://github.com/StellarViewOrg/stellarview-indexer) | Go service that ingests Stellar data into PostgreSQL/TimescaleDB with live event publishing. | |
| [**StellarView TUI**](https://github.com/StellarViewOrg/stellarview-tui) | Keyboard-driven terminal block explorer for investigating Stellar data from the shell. | |
| [**StellarView Docs**](https://github.com/StellarViewOrg/stellarview-docs) | Guides, architecture, and reference for the StellarView suite. | [Live](http://docs.stellarview.acachete.xyz/) |

## Contributing

StellarView is built in the open and welcomes contributions. Check the issues in each repository to find something to work on — from good first issues to deeper features.
